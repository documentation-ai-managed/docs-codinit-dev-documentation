## Release History

<Update label="2024-10-15" description="v2.1.0" tags={["feature", "improvement"]}>

## New Features

- Added support for custom AI model providers beyond OpenAI and Anthropic
- Introduced pipeline visualization tools for better debugging
- Enhanced component library with new AI-powered UI elements

## Improvements

- Improved performance for large-scale deployments
- Better error handling in authentication flows
- Updated documentation with more code examples

</Update>

<Update label="2024-09-20" description="v2.0.0" tags={["feature", "breaking"]}>

## New Features

- Complete rewrite for better modularity and vendor agnosticism
- New component system allowing any framework integration
- Open-source core released under MIT license

## Breaking Changes

- API endpoints have been restructured for better consistency
- Configuration format changed to support multiple providers
- Some legacy components require migration

</Update>

<Update label="2024-08-10" description="v1.5.0" tags={["feature", "bugfix"]}>

## New Features

- Added webhook support for real-time integrations
- New deployment options including self-hosted
- Improved authentication with OAuth providers

## Bug Fixes

- Fixed memory leaks in long-running AI processes
- Resolved issues with model switching
- Corrected deployment failures on certain cloud platforms

</Update>
