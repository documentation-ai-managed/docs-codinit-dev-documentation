## Continuous Deployment

Automate your deployment process with CI/CD pipelines.

<Steps>
  <Step title="Set Up CI" icon="git-branch">
    Configure GitHub Actions or similar CI service.
  </Step>
  <Step title="Add Tests" icon="check-circle">
    Include automated tests in your pipeline.
  </Step>
  <Step title="Configure CD" icon="arrow-right">
    Set up automatic deployment to staging/production.
  </Step>
  <Step title="Monitor Releases" icon="eye">
    Track deployment success and rollback if needed.
  </Step>
</Steps>

<CodeGroup tabs="GitHub Actions,Jenkins">
  ```yaml
name: Deploy to Production
on:
  push:
    branches: [main]
jobs:
  deploy:
    runs-on: ubuntu-latest
    steps:
      - uses: actions/checkout@v2
      - run: npm ci
      - run: npm run build
      - run: npm run deploy
  ```
  ```groovy
pipeline {
  agent any
  stages {
    stage('Build') {
      steps {
        sh 'npm install'
        sh 'npm run build'
      }
    }
    stage('Deploy') {
      steps {
        sh 'npm run deploy'
      }
    }
  }
}
  ```
</CodeGroup>

## Troubleshooting Deployment

Common deployment issues and solutions.

| Issue | Solution |
|--------|----------|
| Build Failures | Check Node.js version and dependencies |
| Environment Variables | Verify all required variables are set |
| Port Conflicts | Ensure ports are available on target server |
| SSL Certificate Errors | Configure certificates properly |
