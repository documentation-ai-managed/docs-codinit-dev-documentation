## Webhook Handling

Implement webhook endpoints to handle real-time events from external services.

<Tabs>
  <Tab title="Receiving Webhooks" icon="download">
    <CodeGroup tabs="Express,FastAPI">
      ```javascript
app.post('/webhook', (req, res) => {
  const event = req.body;
  // Process webhook event
  codinit.processEvent(event);
  res.status(200).send('OK');
});
      ```
      ```python
from fastapi import FastAPI

app = FastAPI()

@app.post("/webhook")
async def handle_webhook(event: dict):
    codinit.process_event(event)
    return {"status": "ok"}
      ```
    </CodeGroup>
  </Tab>
  <Tab title="Sending Webhooks" icon="send">
    <CodeGroup tabs="JavaScript,Python">
      ```javascript
const webhookUrl = 'https://example.com/webhook';
await fetch(webhookUrl, {
  method: 'POST',
  headers: { 'Content-Type': 'application/json' },
  body: JSON.stringify({ event: 'user_created', data: user })
});
      ```
      ```python
import requests

webhook_url = 'https://example.com/webhook'
data = {'event': 'user_created', 'data': user}
response = requests.post(webhook_url, json=data)
      ```
    </CodeGroup>
  </Tab>
</Tabs>

<Callout kind="tip">Always verify webhook signatures to ensure security.</Callout>
