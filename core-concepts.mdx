## Models

Models are the AI brains of your application. CodinIT supports any model provider, allowing you to choose the best tool for each task.

<Tabs>
  <Tab title="Text Models" icon="file-text">
    GPT, Claude, and custom language models for text generation and analysis.
  </Tab>
  <Tab title="Vision Models" icon="eye">
    Image recognition and generation models from various providers.
  </Tab>
  <Tab title="Custom Models" icon="settings">
    Integrate your own trained models via our API.
  </Tab>
</Tabs>

<CodeGroup tabs="JavaScript,Python">
  ```javascript
const model = codinit.model('openai/gpt-4');
const response = await model.generate({
  prompt: 'Explain quantum computing',
  maxTokens: 500
});
  ```
  ```python
model = codinit.Model('anthropic/claude-3')
response = model.chat('What is machine learning?')
  ```
</CodeGroup>

<Callout kind="tip">Experiment with different models to find the best fit for your use case.</Callout>

## Pipelines

Pipelines orchestrate the flow of data through your AI application, connecting models and components in a logical sequence.

<CodeGroup tabs="YAML,JSON">
  ```yaml
pipeline:
  - input: user_query
  - model: nlp_processor
  - component: response_formatter
  - output: final_response
  ```
  ```json
{
  "pipeline": [
    {"step": "input", "type": "text"},
    {"step": "process", "model": "sentiment"},
    {"step": "output", "format": "json"}
  ]
}
  ```
</CodeGroup>

<Callout kind="info">Pipelines enable complex workflows like document analysis followed by summarization.</Callout>
